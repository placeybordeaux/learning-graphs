Depth-First Search

A graph search procedure

  • Overview
  • Excercises

Overview

Depth-first search is a type of graph search. It explores the vertices of a graph. Depth-first search starts at a node in the graph. From there, it explores the rest of the vertices, starting with the start's neighbors.

Nodes that are two edges away from the are explored next, followed by nodes that are three edges away from the start, and so forth.

Pseudo Code

Input: A graph G and a root v of G
Output: A goal node t if it exists.

  1. procedure DFS(G,v):
  2.     create a stack Q
  3.     push v onto Q
  4.     mark v
  5.     while Q is not empty:
  6.         t = Q.pop()
  7.         if t is what we are looking for:
  8.             return t
  9.         for all edges e in G.adjacentEdges(t) do
  10.             u = G.adjacentVertex(t,e)
  11.             if u is not marked:
  12.                 mark u
  13.                 push u onto Q

Complexity

Time Complexity: O(|V| + |E|)
Space Complexity: O(|V|)
